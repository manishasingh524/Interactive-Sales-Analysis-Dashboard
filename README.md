# Interactive-Sales-Analysis-Dashboard
A Powe BI dashboard for analyzing sales , profit and performance across regions and product catergories

## Overview
Interactive Power BI dashboard for analyzing sales and profit performance.

## Tools Used
- Power BI
- DAX
- CSV Dataset

## Key Features
- KPI cards (Sales, Profit, Profit Margin, Quantity)
- Sales by Category and Region
- Profit Trend Analysis
- Interactive slicers

## Insights
- Electronics category drives the highest sales
- West region generates maximum revenue
- Profit varies across months indicating seasonality
